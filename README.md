## 👀 프로젝트 소개
나만의 키보드 쇼핑몰 프로젝트

* 스프링 부트, 스프링 시큐리티를 사용하여 웹 애플리케이션 생애 주기 기획부터 배포 유지 보수까지 전과정 개발과 운영 경험 확보
* JPA, Hibernate를 사용한 도메인 설계
* MVC 프레임워크 기반 백엔드 서버 구축
* 헥사고날 아키텍처
* AWS / 리눅스 기반 CI/CD 무중단 배포 인프라 구축
<br>

## 📅 프로젝트 기간
